# Приложение для расчёта стоимости оборудования.

### Что делает
- рассчитывает стоимость транспорта
- рассчитывает цену для клиента
- Возможно получать другие коэффициенты

## Реализуется через flet

### Код для запуска файла requirements.txt
    pip install -r requirements.txt


### ПЛАНЫ
    Отрицательные значения при вводе
    Определение большего значения как длины
    Структура приложения
    Виртуальное окружение 
    Подключение расчета транспорта к проекту
